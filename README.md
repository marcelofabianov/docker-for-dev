## PHP 8.1+ e 8.2+

Tenho alguns exemplos de Dockerfile para PHP 8.1+ com algumas extensões que utilizo em meus projetos.

Algumas extensões:

- bcmath
- intl
- opcache
- mbstring
- soap
- redis
- apcu
- xdebug
- swoole
- pgsql pdo_pgsql
- oci pdo_oci
- mysql pdo_mysql

- Redis
- PostgreSQL
- Oracle
- MySQL
- SqlServer (Breve)
- Firebird (Breve)
